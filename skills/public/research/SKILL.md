## Research Workflow

1. **Clarify the question** — Break down what specifically needs to be answered
2. **Search broadly** — Use web_search to find multiple sources on the topic
3. **Deep dive** — Use web_fetch to read the most relevant pages in detail
4. **Cross-reference** — Compare information across sources for accuracy
5. **Synthesize** — Combine findings into a clear, structured answer with citations

## Output Format

- Start with a brief summary (2-3 sentences)
- Follow with detailed findings organized by sub-topic
- End with sources list
